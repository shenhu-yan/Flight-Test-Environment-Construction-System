# 飞行试验环境构建系统 - 前端

## 快速启动

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

启动后访问 http://localhost:3001/

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产版本

```bash
npm run preview
```

## 实现功能

| 页面       | 路径           | 功能说明                       |
|-----------|---------------|-------------------------------|
| 登录页     | /             | 用户登录、角色权限控制           |
| 仪表盘     | /dashboard    | 系统概览、统计卡片、ECharts图表  |
| 项目管理   | /projects     | 项目创建、编辑、删除             |
| 环境管理   | /environments | 环境列表、状态筛选、详情查看、删除 |
| 模型管理   | /models       | 模型列表、类型筛选、上传、删除    |
| 环境生成   | /env-gen      | 环境配置生成、模板管理、导入环境   |
| 环境调整   | /env-adjust   | 实时监测仪表盘、自动/手动调整、调整历史、性能对比 |
| 环境优化   | /env-optimize | 环境评估、参数优化、可视化展示、持续优化调度、优化历史、批量优化 |
| 用户管理   | /users        | 用户列表、创建用户（仅管理员可见） |

## 技术栈

- Vue 3 + Vite 5
- Vue Router 4
- ECharts 5
